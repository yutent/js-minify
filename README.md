# es7-to-es5
> 一个简单的将es7语法编译为es5语法的工具。
>> 只编译 es6、es7 后缀名的js文件。

## 配置
> 除设置里的插件配置外, 支持在项目根目录放置`es7toes5.json`, 为每个不同的项目配置不同的编译环境。
>> 支持 `outdir` 配置, 即相对于项目根目录的一个 输出目录。不配置则默认编译保存文件所在目录。
